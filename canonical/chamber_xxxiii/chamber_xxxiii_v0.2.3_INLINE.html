Testing κ₃ (Depth Balance)

  1. Select Tree structure type
  2. Choose Skew Tree for max imbalance
  3. Set L=80, side branches=0.15
  4. κ₃ family with Δ=10
  5. Use Depth Residual mode
  6. Run and observe depth imbalance reduction

τ-Metric Roles

  • Protected (P): Must remain stable (≤0.1% drift). Counts toward ID threshold.
  • Diagnostic (D): Allowed to change. Informational only.
  • For trees: tree_height/paths are diagnostic (κ₃ will change them)
  • entropy/spectral radius are typically protected
